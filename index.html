SEMANTIC TAGS

*Semantic HTML tags are tags that define the meaning of the content they contain.
*Semantic tags are HTML tags that have a specific meaning. For example, the tag is a semantic tag that represents a heading.
The tag is another semantic tag that represents a paragraph. Semantic tags are important because they help to make your HTML code more readable and understandable for both humans and machines.


NON-SEMANTIC TAGS

*They serve only as content holders but give no indication as to what type of content they contain or what role that content plays on the page.
*Non-semantic tags are HTML tags that do not have a specific meaning. For example, the tag is a non-semantic tag that can be used to group elements together.
The tag is another non-semantic tag that can be used to style text. Non-semantic tags are often used for layout purposes, but they can also be used to add structure to your HTML code.
